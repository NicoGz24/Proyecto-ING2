Bienvenido a VACUNASSIT

Desde nuestra plataforma podrá solicitar un turno para poder vacunarse tanto para la Gripe, la Fiebre Amarilla como así también para el Covid19.

Además contamos con 3 sedes (Terminal de Ómnibus - Cementerio - Municipal) para una gestión mas organizada en nuestros turnos.

Para poder acceder a la plataforma será necesario que se registre en el sistema para lo cual deberá ingresar los datos solicitados. Los mismos tendrán carácter de declaración jurada. Una vez registrado podrá iniciar sesión para así, acceder a las funcionalidades de la plataforma de VACUNASSIT.
